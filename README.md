# FaraDeviceDriverChecker

Windowsのデバイスドライバーの状態を診断し、必要に応じてWindows Updateを通じて更新をサポートするGUIツールです。

## 主な機能

- **マルチカテゴリ診断**: オーディオ、ディスプレイ、ネットワーク、Bluetooth、USB、入力デバイス、カメラなど、主要なデバイスカテゴリを個別にスキャン。
- **デバイス状態の可視化**: デバイスマネージャーのステータスに基づき、正常、注意、エラーの状態を分かりやすく表示。
- **ドライバー更新チェック**: Windows Update APIを利用して、利用可能なドライバー更新プログラムを確認。
- **Windows Update連携**: アプリケーション内から直接Windows Updateの設定を開いたり、更新を試行したりすることが可能。

## インストール

インストールは不要です。

1. [最新リリース](https://github.com/fara1991/FaraDeviceChecker/releases)から `FaraDeviceDriverChecker-vX.X.X-win-x64.zip` をダウンロード
2. ZIPファイルを解凍
3. `FaraDeviceDriverChecker.Wpf.exe` を実行

## 使い方

1. **カテゴリの選択**: 左側のメニューから、診断したいデバイスカテゴリ（例：オーディオ）を選択します。
2. **スキャンの実行**: 「スキャン」ボタンをクリックすると、選択したカテゴリのデバイス一覧と状態が表示されます。
3. **更新の確認**: 「更新をチェック」ボタンをクリックすると、Windows Updateで利用可能なドライバー更新があるか確認します。
4. **設定へのアクセス**: 「設定を開く」ボタンから、システムのWindows Update設定を直接開くことができます。

## 結果の見方

各デバイスのステータスに応じて、以下のメッセージが表示されます：

- **✅ 正常に動作しています**: デバイスは正しく認識され、問題なく動作しています。
- **⚠️ 注意**: ドライバーが古い可能性がある、または何らかの構成上の注意点があります。
- **🔴 エラー**: デバイスが正常に動作していないか、ドライバーに不具合が生じている可能性があります。

## システム要件

- **OS**: Windows 10 以降 (64bit推奨)
- **ランタイム**: .NET 9.0 Desktop Runtime
- **権限**: 詳細な情報を取得する場合や更新を試行する場合は、管理者権限での実行を推奨します。

## トラブルシューティング

### プログラムが起動しない
- [.NET 9.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/9.0) がインストールされているか確認してください。
- セキュリティソフトによってブロックされている場合は、実行を許可してください。

### デバイスが表示されない
- デバイスが物理的に接続されているか確認してください。
- デバイスマネージャーでデバイスが認識されているか確認してください。

## 注意事項

- このツールはデバイスの状態を確認し、標準的なWindows Updateの機能を呼び出すものです。
- 診断結果に基づいてドライバーを更新する際は、事前にシステムの復元ポイントを作成することを推奨します。
- 企業環境では、IT管理者に相談してからドライバー更新を行ってください。
